# Настройки и таблицы итогов регистров

Начиная с версии DaJet HTTP Server версии 4.0.9, реализован доступ в запросах DaJet Script к служебным таблицам настроек и итогов регистров.

> **Внимание!** Следует иметь ввиду, что, в зависимости от версии платформы и настроек объектов метаданных, некоторые служебные таблицы или их поля могут отсутствовать.

- Регистры накопления
  - РегистрНакопления.{Имя регистра}.Настройки ```_AccumRgOpt```
  - РегистрНакопления.{Имя регистра}.Итоги ```_AccumRgT``` или ```_AccumRgTn```

- Регистры сведений
  - РегистрСведений.{Имя регистра}.Настройки ```_InfoRgOpt```
  - РегистрСведений.{Имя регистра}.СрезПервых ```_InfoRgSF```
  - РегистрСведений.{Имя регистра}.СрезПоследних ```_InfoRgSL```

- Регистры бухгалтерии
  - РегистрБухгалтерии.{Имя регистра}.Настройки ```_AccRgOpt```
  - РегистрБухгалтерии.{Имя регистра}.ЗначенияСубконто ```_AccRgED```
  - РегистрБухгалтерии.{Имя регистра}.ИтогиПоСчетам ```_AccRgAT0```
  - РегистрБухгалтерии.{Имя регистра}.ИтогиМеждуСчетами ```_AccRgCT```
  - РегистрБухгалтерии.{Имя регистра}.ИтогиПоСубконто1 ```_AccRgAT1```
  - РегистрБухгалтерии.{Имя регистра}.ИтогиПоСубконто2 ```_AccRgAT2```
  - РегистрБухгалтерии.{Имя регистра}.ИтогиПоСубконто[n] ```_AccRgAT[n]```

**Примеры таких запросов**

```SQL
SELECT RegID, Period, ActualPeriod
     , Periodicity, RepetitionFactor
     , UseTotals, UseSplitter, MinPeriod
     , MinCalculatedPeriod, ОбластьДанныхОсновныеДанные
  FROM РегистрНакопления.ТоварыНаСкладах.Настройки
```

```SQL
SELECT TOP 5 Период, Организация
     , Графа4, Графа5, Графа6, Графа7
     , НДС, ДоходЕНВД, РасходЕНВД, ДоходПатент
     , Splitter, ОбластьДанныхОсновныеДанные
  FROM РегистрНакопления.КнигаУчетаДоходовИРасходов.Итоги
```

Какие конкретно поля возвращают те или иные служебные таблицы можно посмотреть, выполнив следующие запросы. Эти запросы также можно выполнять в адресной строке браузера.

#### GET ```/md/entity/{database}/{type}/{name}```

Получает метаданные основной таблицы объекта.

```
curl -X GET http://localhost:5000/md/entity/MS_TEST/РегистрБухгалтерии/Хозрасчетный
```

#### GET ```/md/entity/{database}/{type}/{name}.{service-table}```

Получает метаданные служебной таблицы объекта.

```
curl -X GET http://localhost:5000/md/entity/MS_TEST/РегистрБухгалтерии/Хозрасчетный.Настройки
```
