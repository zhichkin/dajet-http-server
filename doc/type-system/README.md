# Система типов DaJet Script 2.0

Система типов унаследована от предыдущей версии [DaJet Script](https://zhichkin.github.io/dajet-script/data-type-system). Однако имеются некоторые отличия:
- тип данных ```number``` больше не используется, вместо него теперь используются ```integer``` и ```decimal```;
- улучшена работа с типом данных ```union``` (составной тип данных).

**Пример объявления всех поддерживаемых типов данных**

```SQL
DECLARE @Булево          boolean  = TRUE
DECLARE @ЦелоеЧисло      integer  = 12345
DECLARE @ДесятичноеЧисло decimal  = 12.34
DECLARE @ДатаВремя       datetime = '2026-04-12T00:00:00'
DECLARE @Строка          string   = 'Это строка'
DECLARE @Идентификатор   uuid     = '643c6b9d-cacf-4048-11f1-3ce54d7b5bf7'
DECLARE @Ссылка          entity   -- Любая ссылка, полученная из базы данных
DECLARE @СоставнойТип    union    -- Составной тип данных (вспомогательный)
DECLARE @Объект          object   -- Объект, полученный запросом из базы данных
DECLARE @Таблица         array    -- Массив объектов для получения записей из базы данных
```

**Поддерживаемые типы данных при передаче параметров в скрипты по HTTP**

```JSON
{
  "Булево": true,
  "Число": 1234,
  "ДатаВремя": "2026-01-01T12:34:56",
  "Строка": "Это строка",
  "Идентификатор": "1677349A-095F-4488-896F-93425B720FEB",
  "Ссылка": "{123:1677349A-095F-4488-896F-93425B720FEB}"
}
```
