# Выполнение параметризованных скриптов DaJet Script

#### POST ```/script```

Endpoint для выполнения параметризованных скриптов DaJet Script.

Каталог публикации скриптов ```scripts``` должен располагаться в корневом каталоге установки и запуска DaJet HTTP Server. Допускается использовать иерархическую структуру каталога публикации ```scripts```.

Параметры скрипта передаются в виде объекта JSON. Если параметры отсутствуют, то необходимо передать пустой объект ```{}``` без свойств. Поддерживаемые типы данных параметров приводятся ниже.

Результат запроса возвращается как объект JSON или простое значение в зависимости от возвращаемого DaJet Script типа данных.

**Пример тестового скрипта ```test.djs```**

```SQL
DECLARE @Код     string -- Входной параметр для скрипта
DECLARE @Таблица array  -- Результат выполнения запроса

USE 'MS_TEST' -- Зарегистрированная на сервере база данных

  SELECT Ссылка, Код, Наименование
    INTO @Таблица
    FROM Справочник.Номенклатура
   WHERE Код = @Код

END

RETURN @Таблица
```

**Файл тела запроса ```parameters.json```**

```JSON
{
  "Код": "1234567890"
}
```

**Запрос**
```
curl -v -X POST http://localhost:5000/script/test.djs -d @parameters.json -H "Content-Type: application/json; charset=utf-8"
```

**Ответ**
```
{
  "success": true,
  "message": "",
  "result": [
    {
      "Ссылка": "{927:643c709d-cacf-4048-11f1-4f9984f0927d}",
      "Код": "1234567890",
      "Наименование": "Товар 1234567890"
    }
  ]
}
```

#### JSON Schema тела запроса

```JSON
{
  "type": "object"
}
```

#### JSON Schema тела ответа

```JSON
{
  "type": "object",
  "properties": {
    "success": {
      "type": "boolean"
    },
    "message": {
      "type": "string"
    },
    "result": {
      "type": "array",
      "items": { "type": "object" }
    }
  },
  "required": [ "success", "message", "result" ]
}
```

#### Поддерживаемые типы параметров запроса

```JSON
{
  "Булево": true,
  "Число": 1234,
  "ДатаВремя": "2026-01-01T12:34:56",
  "Строка": "Это строка",
  "Идентификатор": "1677349A-095F-4488-896F-93425B720FEB",
  "Ссылка": "{333:1677349A-095F-4488-896F-93425B720FEB}"
}
```
