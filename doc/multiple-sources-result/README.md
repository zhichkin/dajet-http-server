# Сборка данных из нескольких источников данных

В тех случаях, когда требуется собрать данные из нескольких источников данных, можно сделать это одним скриптом. Ниже следующий пример демонстрирует такую возможность.

```SQL
PRIVATE @Таблица1  array  -- Результат выполнения запроса к базе данных MS_UNF
PRIVATE @Таблица2  array  -- Результат выполнения запроса к базе данных PG_UNF
PRIVATE @Результат object -- Результирующий объект для сборки полученных данных

USE 'MS_UNF'
  SELECT TOP 1 Ссылка, Код, Наименование
    INTO @Таблица1
    FROM Справочник.Номенклатура
END

USE 'PG_UNF'
  SELECT TOP 1 Ссылка, Код, Наименование
    INTO @Таблица2
    FROM Справочник.Номенклатура
END

SET @Результат.MS = @Таблица1
SET @Результат.PG = @Таблица2

RETURN @Результат
```

В результате выполнения выше приведённого скрипта будет получен следующий ответ:

```JSON
{
  "async": false,
  "success": true,
  "message": "",
  "result": {
    "MS": [
      {
        "Ссылка": "{76:d0747380-7d2b-8cfd-11e5-47e1f8c9ef49}",
        "Код": "ФР-00000223",
        "Наименование": "Лак мебельный"
      }
    ],
    "PG": [
      {
        "Ссылка": "{76:d0747380-7d2b-8cfd-11e5-47e1f8c9ef49}",
        "Код": "ФР-00000223",
        "Наименование": "Лак мебельный"
      }
    ]
  }
}
```
